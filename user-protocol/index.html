《温暖聊天》用户协议

一、适用范围

  1. 温暖聊天（以下简称“本服务”）是温暖聊天 App（以下简称“本应用”）提供的交流、视频、社交等服务。
  2. 本协议作为本应用用户（以下简称“用户”）与温暖聊天之间用户与服务提供者的法律协议，用户一旦使用本应用，即意味着同意遵守本协议的规定。

二、服务内容

  1. 温暖聊天App为用户提供视频、图片、音频等互联网聊天服务，以及查看相关信息、参加活动、评论等服务。
  2. 用户可通过温暖聊天App发布自己的视频、图片、音频等内容。

  3. 本服务提供的内容包括但不限于：

    3.1 在本应用上发表的文字、图片、视频等；

    3.2 本应用拥有的社交功能，包括但不限于私聊等；

    3.3 本应用拥有的其他功能，包括但不限于语音交流等。

三、免责申明

  1. 本应用提供的服务均仅供用户个人私人使用，不得用于商业目的，否则由此产生的一切责任，由用户自行承担。
  2. 用户使用本应用时，需遵守当地法律法规，不得发表含有违反国家相关法规的内容，否则由此产生的一切后果，由用户自行承担。
  3. 本应用会尽最大努力维护服务的安全性，但由于技术为限，不能完全消除设备的安全风险，用户在使用本应用时，需自行承担风险。

四、个人信息安全

  1. 温暖聊天App将采取业界先进的安全技术，努力确保用户的个人信息安全。
  2. 温暖聊天App将严格遵守国家有关法律法规，对用户个人信息进行隐私保护。
  3. 温暖聊天App将采取商业上合理的措施来保护用户的个人信息，防止其被未经授权的访问、使用或泄露。
  4. 用户的个人信息将仅用于提供温暖聊天App的服务，用户禁止将其他人的个人信息分享给其他人。

五、知识产权

  1. 温暖聊天App上的文字、图片、音频、视频等知识产权，均属于温暖聊天App及其关联公司所有。
  2. 用户可以在遵守本协议的前提下，使用温暖聊天App上的知识产权内容，但不得用于商业目的。

六、用户账号安全

  1. 用户有责任妥善保管注册账号及密码，用户将对注册账号以及密码下发生的所有活动（包括但不限于发布信息、网上点击同意或提交各类规则协议等）承担责任。
  2. 因黑客行为或用户的保管疏忽导致账号、密码遭他人非法使用，本APP不承担任何责任。

七、法律管辖

本协议及温暖聊天App相关事宜，按照中国法律予以解释，并受中国法院管辖。本协议不涉及用户与本App其他用户之间因温暖聊天App服务而产生的任何法律关系或法律纠纷。
